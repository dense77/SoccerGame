# 技术栈明细

## 核心

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 18.x | UI 框架，组件化构建游戏界面 |
| TypeScript | 5.x | 类型安全，减少运行时错误 |
| Vite | 5.x | 构建工具，开发热更新 + 生产打包 |

## 数据层

| 技术 | 版本 | 用途 |
|------|------|------|
| Dexie.js | 4.x | IndexedDB 封装，前端本地数据库 |
| dexie-react-hooks | 1.x | Dexie.js 的 React hooks 绑定 |

## 状态管理

| 技术 | 版本 | 用途 |
|------|------|------|
| useReducer | 内置 | 复杂游戏状态管理（比赛流程、事件队列） |
| useContext | 内置 | 全局状态共享（当前球队、游戏设置） |

## 样式

| 技术 | 版本 | 用途 |
|------|------|------|
| CSS Modules | 内置 | 组件级样式隔离 |

## 工具

| 技术 | 版本 | 用途 |
|------|------|------|
| ESLint | 9.x | 代码规范检查 |
| Prettier | 3.x | 代码格式化 |

## 部署

| 平台 | 说明 |
|------|------|
| GitHub Pages | 免费静态托管，直接关联仓库 |

## 未来扩展（按需引入）

| 技术 | 用途 |
|------|------|
| React Router | 多页面路由（球队管理 / 比赛 / 设置） |
| TanStack Query | 服务端状态管理，API 请求缓存 |
| Framer Motion | 交互动画（进球特效、转场） |
| Tailwind CSS | 原子化样式，快速构建 UI |
