# SoccerGame

A football text-based simulation game. Control matches through button interactions and experience the life of a football manager.

## Tech Stack

- React 18 + TypeScript
- Vite
- Dexie.js (IndexedDB)

## Getting Started

```bash
# Clone the repo
git clone https://github.com/dense77/SoccerGame.git
cd SoccerGame

# Install dependencies
npm install

# Start dev server
npm run dev
```
